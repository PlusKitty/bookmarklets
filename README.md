# Bookmarklets
Resource hungry browser extensions? Not today! Try bookmarklets
## Adding bookmarklets
Go to "Add bookmark" and paste the bookmarklet code as the URL.
## Using bookmarklets
Go to your desired webpage and hit the bookmark. JS magic!
## Index
* [xkcd-transcript](https://github.com/PlusKitty/bookmarklets/blob/main/xkcd-transcript.js): show the descriptive transcript of xkcd comics on the site.
* [xkcd-transcript-get](https://github.com/PlusKitty/bookmarklets/blob/main/xkcd-transcript-get.js): download HTML with the image link, including on-hover text, and visible transcript. NOTE: image will be visible only if you can connect to the xkcd website.
